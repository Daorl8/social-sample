# STRUCTURE — ROUND 라운드 (social-sample)

단일 파일 정적 사이트. `index.html`에 HTML/CSS/JS 인라인.

## 무드/축 (매트릭스)
- 무드(F): **웜 소프트팝**(웜화이트 + 코랄 + 플럼, 라운디드·친근) — 기존 12종에 없던 밝고 친근한 트렌디 무드. 베이지 클러스터·다크와 구별.
- 페이지모델(A): **모집/신청-퍼스트**(회차 일정 리스트 + 참가 신청) — 신규 아키타입(이벤트/코호트 모집형)
- 주인공(B): 모임 신청 전환 + 신뢰(인증)
- 시그니처(E): **인터랙티브 로테이션 다이어그램**(자리가 도는 시각화) — 신규 인터랙션
- 니치: 유료 신청형 소셜 만남(로테이션 소개팅, 2025~26 최트렌디)

## 섹션
1. 상단바(sticky) — 브랜드·내비·참가신청 CTA
2. 히어로 — "하루 저녁, 10명과 10분씩" + 신뢰 배지(인증·성비·누적)
3. 진행 방식 — 로테이션 다이어그램 + 4스텝(신청·인증→10분 로테이션→애프터→매칭)
4. 안심/인증 — 직장·미혼·본인 인증, 성비 1:1, 운영 규칙
5. 다가오는 회차 — 날짜·지역·성비·연령·잔여석(마감임박/모집중) + 회차별 신청
6. 테마 라운드 — 와인·보드게임·한옥·취향 매칭
7. 후기·매칭률 — 누적 매칭·만족도·재참여율 + 후기
8. 참가 신청 폼 — 공통 핸들러(데모 성공/실ID 2xx), 정직 라벨 "참가 신청"
9. FAQ — 아코디언
10. 모바일 하단바 — 참가 신청 + 카카오(PC 메인 CTA 일치)
